1. Download xampp dan pastikan menggunakan versi PHP anatara 5.5 sampai 7.4.0
2. Install xampp di PC / Komputer Anda
3. Jalankan xampp dengan klik shortcut xampp kemudian buat Apache & MySQL dalam kondisi start
4. Buka browser dan ketik localhost/phpmyadmin kemudian enter
5. Setelah berhasil pilih menu database atau basis data kemudian buat database dengan nama pos-kasir-grosir-eydcom.com
6. Setelah berhasil buat database pilih menu impor dan arahkan ke folder hasil ekstrak dan masuk 
folder database dan pilih pos-kasir-grosir-eydcom.com.sql
7. Klik tombol kirim dan tunggu prosesnya sampai selesai
8. Versi XAMPP yg disarankan untuk Operasi System Windows
